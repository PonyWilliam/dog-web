### 虹软人脸采集sdk

1、插件说明
本插件时基于虹软免费版人脸识别sdk3.1版本开发，如需使用此插件，请在虹软开发平台申请此版本sdk权限

目前仅支持android版本，iOS版本待后续推出。

2、使用步骤

1）在虹软开发平台创建应用，并给应用添加人脸识别SDK权限

2）引用插件后，需配置虹软app_id 和 虹软sdk_key
